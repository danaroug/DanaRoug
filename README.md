## Featured projects

### [TalesOfCoding](https://www.talesofcoding.com) — ASP.NET Core MVC platform

TalesOfCoding is an independently developed technology publishing platform built with ASP.NET Core MVC, C#, Entity Framework Core, SQL Server, Bootstrap, ASP.NET Core Identity, and Azure.

The platform combines public content discovery with authenticated publishing and account-management workflows for creators.

Selected engineering work:

* Implemented registration, email confirmation, authentication, account-management, personal-data, and account-deletion workflows
* Added role-based administration and user-management functionality
* Built blog and tutorial publishing workflows with ownership-based authorisation
* Separated public catalogue pages from authenticated creator-management workflows
* Added category-based content discovery across blog posts and tutorials
* Built a public search feature for finding published blog posts and tutorials
* Added automated tests for access-control and ownership behaviour
* Deployed and maintained the application using Azure App Service, Azure SQL, and a custom domain
* Integrated authenticated transactional email delivery

TalesOfCoding is in active development. As an independently funded project, public access may occasionally be paused during development, testing, or infrastructure maintenance.

The source repository is private, but the live application is available at [talesofcoding.com](https://www.talesofcoding.com).

### [FoxiTrack](https://github.com/danaroug/FoxiTrack) — ASP.NET Core and React application

FoxiTrack is a work-in-progress full-stack job application tracker designed to organise roles, companies, application stages, interviews, and notes within a single workflow.

The project uses:

* ASP.NET Core Web API
* React
* Entity Framework Core
* SQL Server
* Component-based frontend development
* Separation between backend and frontend responsibilities

Current development work focuses on application structure, API integration, frontend state management, and expanding the project into a complete job-search management tool.
